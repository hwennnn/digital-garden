|**Version**|**Key Features**|
|---|---|
|**HTTP/1.1**|Keep-Alive, pipelining (rarely used)|
|**HTTP/2**|Multiplexing, header compression, binary framing|
|**HTTP/3**|Uses QUIC (UDP-based), reduces latency, better for mobile networks|
